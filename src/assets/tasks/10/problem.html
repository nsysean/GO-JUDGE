Let's watch critically
time limit per test
2.5 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

You can now finally take a break and watch videos from Let's Think Critically! There are $$$n$$$ videos in total with indices 1 to $$$n$$$ and length $$$l_1, l_2, l_3 ... l_n$$$ minutes respectively. Your break is $$$x$$$ minutes long and you want to fully use this time (i.e. watch exactly $$$x$$$ minutes of video, no more no less). Also, you are too tired to do anything so you just want to choose a video (index $$$k$$$, $$$k\leq n$$$) to watch and keep on watching until you finish a video, then you can stop or continue watch video $$$k+1$$$ (you can only watch the next video if and only if $$$k+1\leq n$$$). You want to find the number of different ways to watch consecutive video(s) such that you fully spend all of your time. Since you are a big fan of Let's Think Critically, you will never pause a video nor quit watching in the middle of it, meaning that if you watch video $$$k$$$, you will watch all $$$l_k$$$ minutes.

It is also possible that there is no such way to fully spend all $$$x$$$ minutes. In this case, output $$$0$$$. You are too tired to do math during the break so you want to make a program that helps you answer this question every time you have a break. Time to watch more videos :D.

Input

The first line consists of 2 integers $$$n$$$ and $$$x$$$. ($$$1\leq n \leq 10^6$$$ and $$$1\leq x \leq 10^{18}$$$)

The second line consists of $$$n$$$ integers with the $$$i$$$th integer representing $$$l_i$$$, the length of video $$$i$$$. (for all $$$1 \leq i \leq n$$$, $$$1\leq l_i \leq10^{12}$$$)

Output

Output a single integer, the number of ways to fully spend all $$$x$$$ minutes

Scoring

Subtask 1 ($$$50$$$pts): for all $$$1 \leq i \leq n$$$, $$$l_i = 1$$$

Subtask 2 ($$$100$$$pts): $$$1 \leq n \leq 100$$$

Subtask 3 ($$$250$$$pts): $$$1 \leq n \leq 10^4$$$

Subtask 4 ($$$600$$$pts): no additional constraints

Example

Input
7 3
2 1 2 5 2 1 3
Output
4
